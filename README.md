# OSSCCU Website

## Project Overview
OSSCCU Website is a platform designed to showcase the OSSCCU community, its projects, and contributions.

## Tech Stack
- **TypeScript**: 91%
- **CSS**: 4%
- **JavaScript**: 2%

## Getting Started

### Prerequisites
- Node.js
- npm (Node Package Manager)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ossccu/ossccu-website.git
   ```
2. Navigate into the project directory:
   ```bash
   cd ossccu-website
   ```
3. Install the dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   # or
   bun install
   
   ```
